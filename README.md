# OTE-sfb1436
This repository contains the code used to perform the analyses and generate the results presented in the paper titled "The neural signature of simple overlearned temporal expectations resembles episodic retrieval" (https://doi.org/10.64898/2026.07.07.737020).

# Data Availability
The scripts in this repository require the corresponding behavioral and fMRI data, which are publicly available via OSF (https://osf.io/am6y7/).

The OSF repository contains:
* Behavioral `.tsv` files required for the behavioral analyses.
* Preprocessed fMRI data required for the first-level analyses.
* First-level contrast images (`con_*.nii`) used as input for the ROI extraction and group-level analyses.
