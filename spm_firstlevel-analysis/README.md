## First-Level Analysis

The first-level fMRI analyses were performed using **SPM12** (Statistical Parametric Mapping, Wellcome Centre for Human Neuroimaging, London, UK).

For all analyses, the general linear model (GLM) included the canonical hemodynamic response function (HRF), its **temporal derivative**, and its **dispersion derivative** as implemented in SPM12.

The first-level analysis pipeline was largely identical for both the **naïve** and **trained** participant datasets. The only exception concerns the trained dataset: because each participant completed three fMRI sessions, all sessions were modeled jointly within a single first-level GLM. In contrast, participants in the naïve dataset completed only a single fMRI session.

The scripts contained in this directory reproduce the complete first-level analyses used to generate the subject-specific contrast images for the group-level analyses.

## Input Data

The first-level analysis requires the **preprocessed fMRI data** generated by the fMRIPrep preprocessing pipeline as input.

The complete preprocessed fMRI dataset for the **naïve** participant group is publicly available via OSF: https://osf.io/am6y7/
Due to storage limitations, the complete preprocessed fMRI dataset for the **trained** participant group is not publicly available via OSF. Researchers interested in accessing these data for non-commercial research purposes are invited to contact **Linda Sempf** at **linda.sempf@ovgu.de**, providing a brief rationale for their request. Access to the requested data will be granted upon review.
